# TrainMe

TrainMe ar en enkel Python-hemsida for traningshjalp med tre spar:

- Atlet
- Aktiv
- Komma igang

Varje spar har subkategorierna styrka, blandat och kondition. Besokare kan anvanda sidan utan konto, men AI-hjalp och Strava-baserade insikter ar lasta bakom konto.

## Starta lokalt

```powershell
python app/main.py
```

Öppna sedan `http://127.0.0.1:8000`.
